## Control flow

### Conditional

```go
if age > 18 {
    fmt.Println("Adult")
} else {
    fmt.Println("Minor")
}

switch day {
    case "Monday":
        fmt.Println("Start of the week")
    default:
        fmt.Println("Other day")
}
```

### Loops

```go
for i := 0; i < 5; i++ {
    fmt.Println(i)
}
```
