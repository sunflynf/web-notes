## Setup evironment on Visual Studio Code

- [Official PHP website](https://www.php.net/downloads)
- Free IDE - [Visual Studio Code](https://code.visualstudio.com/docs/languages/php)
- Paid IDE - [PHPStorm](https://www.jetbrains.com/phpstorm/)

### Configure PHP Debug

```json title='launch.json'
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Listen for XDebug",
      "type": "php",
      "request": "launch",
      "port": 9003,
      "pathMappings": {
        "/var/www/html": "${workspaceFolder}"
      }
    },
    {
      "name": "Launch currently open script",
      "type": "php",
      "request": "launch",
      "program": "${file}",
      "cwd": "${fileDirname}",
      "port": 9003
    }
  ]
}
```
