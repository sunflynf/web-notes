## Colours

| Type                                                      | Example                      |
| --------------------------------------------------------- | ---------------------------- |
| [Name](https://www.w3schools.com/colors/colors_names.asp) | red, black, white, blue, ... |
| **RGB**                                                   | rgb(0, 0, 0)                 |
| RGBA                                                      | rgba(0, 0, 0, 0.2)           |
| **HEX**                                                   | #000000                      |
| HSL                                                       | hsl(0, 100%, 100%)           |
| HSLA                                                      | hsl(0, 100%, 100%, 0.2)      |

:::info HSL colors

- **hue** `[0, 360]`
  - 0 - red
  - 120 - green
  - 240 - blue
- **saturation**
  - 0% - shade of gray
  - 100% - full color
- **lightness**
  - 0% - dark
  - 50% - neither light or dark
  - 100% - light

:::

:::tip Special colors

- `transparent`: like `opacity`
- `currentcolor`: get initial color of element, use with pseudo

:::
