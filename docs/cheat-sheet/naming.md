# Naming styles

## **hy-phens**

:::info

- **HTML**'s attributes - id, class
  - `form-group`
  - `form-check`
  - `form-label`
- folder name
  - `partial-field`
  - `<field-name>-<features>`

:::

## **under_score**

:::note

- Database variables
  - `user_id`
  - `user_name`
  - `total_price`
- _Optional_: use for variables in Code (languages allow)

:::

## **UpperCamelCase (PascalCase)**

:::info

- `class` in OOP languages
  - `Account`
  - `Product`
  - `User`
- **React**'s components
  - `Modal`
  - `Button`
  - `ButtonGroup`
  - `Sidebar`

:::

## **lowerCamelCase**

:::note

- variable
  - `firstName`
  - `lastName`
  - `address`
- function:
  - `getProducts`
  - `setProduct`
  - `getTotalPrice`

:::
